## 전체 설명 (Full description, ≤4000자)

**한국어**

CleanCAD Viewer는 건설 현장에서 쓰기 좋은, 광고 없는 DWG 도면 뷰어입니다.
무겁고 광고 가득한 뷰어 대신, 빠르고 깔끔하게 도면만 봅니다.

■ 주요 기능
• AutoCAD DWG 파일을 기기에서 바로 열어 보기
• 카카오톡·이메일·파일 관리자에서 받은 .dwg를 "열기"로 바로 표시
• 부드러운 손가락 확대/축소·이동
• 한글·일본어·중국어 문자 정상 표시
• 도면별 시트 자동 인식 + 탭 전환
• 선·치수·해치·블록·문자 등 도면 요소 렌더링
• 어두운 화면(다크 모드) 지원
• 최근 파일 빠른 열기

■ 깨끗함에 진심입니다
• 광고 없음
• 계정/로그인 없음
• 인터넷 통신 없음 — 완전 오프라인 동작
• 요청하는 권한 0개 — 개인정보를 수집하지 않습니다

■ 오픈소스
CleanCAD Viewer는 GPL v3 오픈소스입니다. 소스 코드는 GitHub에 공개되어 있습니다.
github.com/june690602-blip/opendwg

도면 파싱에는 오픈소스 라이브러리 LibreDWG(GPL v3)를 사용합니다.

문의: june690602@gmail.com

**English**

CleanCAD Viewer is an ad-free DWG drawing viewer built for construction-site use.
Instead of heavy, ad-filled apps, it just shows your drawings — fast and clean.

■ Features
• Open AutoCAD DWG files directly on your device
• Open .dwg shared from messengers, email, or a file manager via "Open with"
• Smooth pinch-to-zoom and pan
• Correct rendering of Korean, Japanese, and Chinese text
• Automatic sheet detection with tab switching
• Renders lines, dimensions, hatches, blocks, text, and more
• Dark mode
• Quick access to recent files

■ Truly clean
• No ads
• No account or sign-in
• No internet access — fully offline
• Zero permissions requested — collects no personal data

■ Open source
CleanCAD Viewer is GPL v3 open source. Source code is public on GitHub:
github.com/june690602-blip/opendwg

Drawing parsing uses the open-source library LibreDWG (GPL v3).

Contact: june690602@gmail.com
